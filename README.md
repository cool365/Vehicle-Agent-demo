# 智驾卫士 AI Copilot

**SafeDrive AI Guardian - Context-Aware Intelligent Cockpit System**

> 懂你所需，更懂安全边界 —— 新一代情境感知智能座舱系统

---

## 🎯 项目简介

智驾卫士是新一代情境感知智能座舱系统，颠覆传统"盲目服从"的车载助手理念。

**核心创新：**
- ✅ **主动安全防护**：不是被动响应，而是主动评估风险
- ✅ **情境感知智能**：同一指令，不同场景下有不同处理策略
- ✅ **透明AI推理**：每个决策都展示完整推理过程
- ✅ **渐进式安全**：分级处理，提供最优交互方案

系统通过实时监控车速、档位、驾驶员分心等级等多维数据，构建动态安全决策模型。当驾驶员分心时主动拦截娱乐请求，高速行驶时建议语音操作，倒车时自动启动辅助系统。

---

## 🚗 核心功能

### 1. 多维场景感知
- 实时监控车速、档位、安全等级
- 检测驾驶员分心状态
- 动态调整AR-HUD透明度

### 2. 智能决策引擎
- 基于规则的风险评估模型
- 功能分类：必需、允许、建议、禁止
- 毫秒级响应时间

### 3. 透明决策过程
- 显示AI推理逻辑
- 展示风险评估结果
- 说明采取的行动

### 4. 数据持久化
- 记录所有车辆状态
- 追踪AI决策历史
- 支持事后审计分析

## 🛠️ 技术栈

**前端：**
- React 18 + TypeScript
- Vite（构建工具）
- Tailwind CSS（样式）
- Lucide React（图标）

**后端：**
- Supabase Edge Functions（Serverless）
- PostgreSQL（数据库）
- Row Level Security（数据安全）

**特性：**
- 实时数据同步
- 中英双语支持（i18n）
- 响应式设计
- 性能监控
