# 智能座舱情境感知决策系统

**Context-Aware Decision System for Smart Cockpit (CADS-Cockpit)**

> 座舱部门技术演示 v1.0 | 多维场景感知 + 智能决策引擎 + 透明AI推理

---

## 🎯 项目简介

本系统是座舱部门开发的情境感知智能决策技术演示，核心创新在于**突破传统"盲目服从"的车载助手理念**，构建基于多维场景感知的主动安全决策模型。

**核心技术特点：**
- ✅ **多维场景感知**：实时监控车速、档位、驾驶员状态等多维数据
- ✅ **智能风险评估**：基于规则引擎的动态安全评估模型
- ✅ **透明决策推理**：展示完整AI决策过程，可解释、可追溯
- ✅ **渐进式交互**：分级处理用户请求，提供最优交互方案

系统通过实时分析驾驶场景，主动判断操作风险等级。当驾驶员分心时自动拦截娱乐请求，高速行驶时建议语音操作，倒车时主动启动辅助系统，实现"懂场景、知风险、能决策"的智能座舱交互。

---

## 🚗 核心功能

### 1. 多维场景感知引擎
- 实时监控：车速、档位、安全等级
- 状态检测：驾驶员分心程度评估
- 动态调整：AR-HUD透明度自适应

### 2. 智能决策引擎
- **风险评估模型**：基于规则的多维度风险计算
- **功能分类系统**：必需/允许/建议/禁止四级分类
- **毫秒级响应**：实时决策，无感知延迟

### 3. 透明决策展示
- **推理逻辑可视化**：显示AI思考过程
- **风险评估结果**：展示安全等级判定
- **行动说明**：解释采取的决策和理由

### 4. 数据持久化与审计
- 记录所有车辆状态变化
- 追踪AI决策历史
- 支持事后分析和模型优化

## 🛠️ 技术架构

**前端技术栈：**
- React 18 + TypeScript
- Vite（构建工具）
- Tailwind CSS（样式系统）
- Lucide React（图标库）

**后端技术栈：**
- Supabase Edge Functions（Serverless 计算）
- PostgreSQL（数据持久化）

**系统特性：**
- 实时数据同步
- 中英双语支持（i18n）
- 响应式设计
- 性能监控面板

## 🎬 演示场景

### 场景1：停车状态（安全场景）
- **系统判断**：车辆静止，无安全风险
- **系统行为**：全功能开放，所有操作均可执行

### 场景2：驾驶中分心（中等风险）
- **系统判断**：驾驶员注意力分散
- **系统行为**：拒绝娱乐请求，保护驾驶专注度

### 场景3：分心 + 紧急导航（智能权衡）
- **系统判断**：必需功能优先级高于分心风险
- **系统行为**：允许导航，自动切换语音模式

### 场景4：倒车状态（主动辅助）
- **系统判断**：特殊操作场景
- **系统行为**：自动启动倒车影像，娱乐功能静音

### 场景5：高速行驶（交互优化）
- **系统判断**：手动操作不便
- **系统行为**：建议语音操作，优化交互方式
